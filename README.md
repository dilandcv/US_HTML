# 🌐 Portafolio Web - Dilan Chávez

Este es un proyecto de portafolio personal desarrollado con **HTML, CSS y JavaScript**, donde se presenta información sobre mí, mis proyectos, mis mascotas y algunas funcionalidades interactivas.

---

## 📌 Descripción

El sitio web está diseñado como una página personal que incluye:

- Información sobre mí  
- Proyectos desarrollados  
- Sección de mascotas  
- Interactividad con JavaScript  
- Formulario de contacto  

El objetivo principal es practicar y demostrar habilidades en desarrollo frontend.

---

## 🛠️ Tecnologías utilizadas

- **HTML5** → estructura del sitio  
- **CSS3** → estilos y diseño responsive (Flexbox y Grid)  
- **JavaScript** → interactividad y manipulación del DOM

## ⚙️ Funcionalidades

### 🧠 Interactividad (JavaScript)

- Mensaje de bienvenida al cargar la página  
- Cambio dinámico de texto  
- Mostrar / ocultar habilidades  

### 📱 Diseño responsive

- Adaptado a móvil, tablet y escritorio  
- Uso de **Grid** para la galería  
- Uso de **Flexbox** para layout  

---

## 🐾 Sección de mascotas

Galería de animales con descripciones personales, usando tarjetas con efecto hover y diseño en grid.

---

## 📬 Formulario de contacto

Campos incluidos:

- Nombre  
- Apellido  
- Correo electrónico  

> Nota: actualmente no tiene conexión a backend.
